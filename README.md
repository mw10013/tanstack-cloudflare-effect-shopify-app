# tanstack-cloudflare-effect-shopify-app

Port of the Shopify App Template to TanStack Start + Cloudflare Workers + Effect v4.

## Local development

```bash
pnpm i
cp .env.example .env
pnpm d1:reset
pnpm shopify:dev
```

## Commands

```bash
pnpm typecheck
pnpm lint
pnpm test
pnpm test:integration
pnpm test:browser
pnpm test:e2e
```
